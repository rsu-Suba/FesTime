## 最優先事項

*   **Supabase のプラン:** 本番運用では Supabase Pro プランを推奨します。無料枠では接続数制限により当日アプリが停止するリスクがあります。
*   **パスの扱い:** `/FesTime` 等のサブディレクトリで運用する場合、画像やリンクには必ず `getPath` 関数を使用してください。
*   **データのバックアップ:** 投票データやユーザー投稿データは Supabase 上にあります。終了後は速やかにエクスポートすることを推奨します。
